# KG-to-Rule Extraction Prompts

This directory stores the prompts used to convert KG evidence into structured rule-library records.

- `aviation_kg_to_rule_prompt.txt`: aviation KG-to-rule extraction prompt.
- `architecture_kg_to_rule_prompt.txt`: architecture KG-to-rule extraction prompt.
- `generic_kg_to_rule_prompt.txt`: earlier generic KG-to-rule extraction prompt.

The prompts define the extraction task, required rule-record schema, provenance fields, constraint fields, inter-rule relation fields, and JSON formatting requirements.
